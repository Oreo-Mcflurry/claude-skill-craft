## Auto-Detection: Skill Creation & Review

When the user asks about creating, improving, or reviewing Claude Code skills, **automatically** invoke the appropriate skill-craft command.

### Detection Triggers

- Creating skills: "스킬 만들어줘", "create a skill", "new skill", "SKILL.md 작성"
- Reviewing skills: "스킬 리뷰", "review my skill", "check this skill", "스킬 개선"

### Commands

- `/skill-create <description>` — Create a new skill following Anthropic's 5 body rules
- `/skill-review <path or name>` — Review an existing skill and get a scored report
